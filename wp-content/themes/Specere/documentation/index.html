A) HTML Structure - top

This theme is a fixed layout. All of the HTML code carries comments to note the start and end of a section, a comment is text held between these tags. The general template structure is the same throughout the template. Here is the general structure.

B) CSS Files and Structure - top

There are 4 CSS files in this theme.

  • reset.css - This file resets browser default styling ready for the global stylesheet. Location: /css/reset.css
  • typo.css - This file sets the default typography. Location: /css/typo.css
  • global.css - This file sets the site styling, this file overrides any of the above. Location: /css/global.css
  • ie.css - This file sets any seperate styling for Internet Explorer 7+. Location: /css/ie.css

All the stylesheets are separated into sections using comments like below:

	/* Header Section ------------------------*/

	some code

	/* Main Section ------------------------*/
	
	some code
	
	/* Sidebar Section ------------------------*/
	
	some code
	
	/* Footer ------------------------*/
	
	some code
	
	etc, etc.

If you would like to edit a specific section of the site, simply find the appropriate label in the CSS file, and then scroll down until you find the appropriate style that needs to be edited.

D) Theme Installation - top

There are two ways to install your new Wordpress theme. You may install via the Wordpress Admin section, or via FTP.

Installation via Admin:

  1. Log in to your Wordpress Admin section at http://your-website.com/wp-admin/
  2. Once logged in you will need to navigate to Appearance > Themes > Install Themes from the left menu. From there click Upload from the top sub-menu.
  3. Click on the Browse button and browse to where you saved your theme and select it. Now click Install Now. Once installed it will return and ask you if you would like to activate the new theme, click activate.

Installation via FTP:

  1. Open up your favourite FTP client and enter your FTP credentials given to you by your Web Host.
  2. Once you have connected to your server via FTP you need to navigate to /wp-content/themes/
  3. Next extract / unzip your theme.zip. This will create a folder with the same name as the theme.zip file only without the .zip extension.
  4. Now you need to upload the extracted folder into your /wp-content/themes/ directory with your FTP client.
  5. Once uploaded, head over to your Wordpress Admin section in your web browser and from the left menu navigate to Appearance > Themes. You will now see your newly installed theme. Click activate.

E) Fullscreen Gallery - top

This theme is primarily focused on showcasing your work, hence the gallery being the home page and the first thing your visitors will see, leaving a great first impression!

This walkthrough will explain exactly how to setup not only the fullscreen gallery, but also the other sections of the site to create a fully functioning showcase like the theme demo.

  • 1) Create a page of your choice, and while inside the page editor under the attributes menu on the right hand side, simply set the page template to 'gallery'. This will setup the page ready for adding your images.
  • 2) Now simply add all of your images either with a definitive URL or via the Image uploader underneath the page editor.
  • Once the images have uploaded click on 'Save All Changes' from the uploader window, then select the image you'd like to use, or you may click on the 'Select All' link and then 'Inert All Images'. Now Publish.
  • 3) That's it! Everything else is done for you. The thumbnails will be autogenerated along with the scrolling navigation.
  • 4) — To choose a static image behind a page simply click on the Featured Image box in a page and upload your image.

F) Theme Options - top

Once you have finished installing your new theme you can then begin editing the look and feel and also the other various theme options from the in-built Options Panel. Once logged into your Wordpress Admin, the theme options panel may be accessed by clicking on the Appearance > Options. This will then open the theme options panel for configuration.

Each tabbed section carries various options which all have descriptions to help you thorough the setup process. We will run through the available settings below.

  • General Settings:
    • Logo — Upload a logo of your choice, if you do not add a logo this will simply default back to the name of your site.
    • Favicon — Upload a 16 x 16px .gif or .png image to be used as your site favicon.
    • Email Address — Set the email address that you would like all mail coming from the in-built contact form to go to.

    Preloader Settings:
    • Preloader Color — Choose a color for the preloader.
    • Disable — Disable the preloader.

    CSS Settings:
    • Custom CSS — Add any additional CSS styling. This will override any other CSS through the site.

    Typography Options:
    • Main Text — Choose the settings for the main body text. Size, type, weight and color.
    • Heading Colors — Choose the color for the page Headings.
    • Cufon Font Replacement — Choose the font you would like for the site headings. If this is set to disabled it will default back to CSS browser fonts.

    Fullscreen Imagery:
    • Disable Panning — Disable the image panning.
    • Disable Overlay — Disable the cross hatch overlay that sits over the top of images. This has the added benefit of preventing right click view image in a browser.

    Social Integration:
    • Twitter — Add your Twitter username. If this is set it will enable the Twitter icon in the navigation and also the Twitter widget on the homepage
    • Facebook — Add your Facebook profile ID. This is the long number at the end of your Facebook URL. i.e: http://www.facebook.com/#!/profile.php?id=1234567890 Paste ONLY the number. If this is set it will enable the Facebook icon in the navigation

    Twitter Widget:
    • Twitter — Add your Twitter username.
    • Tweet Number — How many Tweets to display?
    • Widget Title — Title to display above the widget.
    • Disable — Disable the widget.

    Flickr Widget:
    • Flickr — Enter your Flickr RSS URL. You can find your Flickr feed by going to Flickr.com, log in and then click Your Photostream. After the page loads, scroll down to the bottom and find Subscribe to Your Name's photostream then copy the link.
    • Flickr Number — How many photos to display?
    • Widget Title — Title to display above the widget.
    • Disable — Disable the widget.

    Popular Widget:
    • Widget Title — Title to display above the widget.
    • Disable — Disable the widget.

    Categories Widget:
    • Widget Title — Title to display above the widget.
    • Disable — Disable the widget.

    Archives Widget:
    • Widget Title — Title to display above the widget.
    • Disable — Disable the widget.

    Blog Options:
    • Static Image — Choose the image you would like to show as the Blog background.
    • Default Display Setting — Set how you would like your posts displayed. Either the full content, or the post excerpt.

    Analytics:
    • Tracking — Enter your Google Analytics or other tracking code.

    SEO:
    • Description and Keywords — Describe your website and services and add relevant keywords that match the description and content.

G) Theme Shortcodes - top

This theme comes with various shortcodes that enable you to create items such as columns, lists and boxes with ease. Shortcodes simply wrap your content in a page or post.

All of the shortcodes can also be seen in action in the live demo under the Features tab.

  1. Available Columns

    — one_full, one_half, one_third, two_third, one_quarter.

    Note: each set of columns with the exception of one_full has a right column with no margin added. This is to be used for the last column to the right. Each right column is simply ended with _last, I.E: one_third_last, one_quarter_last etc...

    Example: 
    [one_full]You would type these shortcodes when adding or editing a post or page.[/one_full]

  2. Accordion

    — accordion.

    Example: 
    [accordion title="Title to be shown on accordion"]accordion content initially hidden...[/accordion]

  3. Toggler

    — toggle.

    Example: 
    [toggle]add your toggle content initially hidden...[/toggle]

  4. Available Boxes

    — default_box, notice_box, success_box, warning_box, download_box.

    Example: 
    [default_box]add your content here, inside the shortcodes...[/default_box]

    5. H) Using the in-built Contact Form - top

    This theme comes with a simple in-built contact form which is easy to setup.

    To use the form simply create a new page from your Wordpress Admin and add any content you wish. Then from the right hand panel while inside the page editor you will see where it says 'Page Attributes' from here you can select a page template to use. Select the 'Contact' page template and click update to save.

    Note: The contact form uses the email address which is set in the theme options panel as described above.
